## Installation

1. Clone the repository:  
   https://github.com/Arkorede/crypto-checkout

2. Navigate to the project directory:  
   cd crypto-checkout

3. Install dependencies:  
   npm install

## Usage

To run the application in development mode:
npm run dev

The application will start and can be accessed at `http://localhost:3000` (or another port if 5173 is in use).

The application is also live on: https://crypto-checkout-psi.vercel.app/

- Next.js
